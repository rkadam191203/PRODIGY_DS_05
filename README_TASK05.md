# 🚗 PRODIGY_DS_05

## US Traffic Accident Data Analysis

### Objective
Analyze traffic accident data to discover patterns related to road conditions, weather, time of day, and accident severity.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Workflow
1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis
5. Visualization
6. Insights

## Visualizations
- Severity Distribution
- Accidents by Hour
- Accidents by Day
- Top States
- Weather Conditions
- Correlation Heatmap
- Temperature vs Severity
- Visibility vs Severity
- Accident Hotspots

## Key Findings
- Peak traffic hours show higher accident frequency.
- Weather and visibility influence accident severity.
- Geographic hotspots help identify high-risk regions.
